# 🏗️ G3-Architects

**G3-Architects** is a modern and responsive landing page built using only **HTML5** and **raw CSS3**.
This project focuses on clean structure, layout design, and responsive styling without using any CSS frameworks.

---

## 📌 Project Overview

G3-Architects is a static landing page designed for an architecture company.
It highlights company information, services, features, and team members with a clean and professional UI.

This project was built to strengthen fundamental front-end skills using pure HTML and CSS.

---

## ✨ Features

* 🏢 Hero section with headline and call-to-action button
* 📖 About section
* 🛠️ Features / Services section
* 👷 Team showcase section
* 📊 Statistics section
* 📱 Fully responsive design
* 🎨 Clean and modern layout

---

## 🛠️ Technologies Used

* **HTML5** – Semantic structure
* **CSS3** – Styling and layout
* **Flexbox & Grid** – Layout alignment
* **Media Queries** – Responsive design

## 💻 How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Double-click on `index.html`.
4. The landing page will open in your browser.

No frameworks or dependencies required.
